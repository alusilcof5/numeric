ℕ
00 — Naturales

Números Naturales

Los primeros números que aprendemos. Surgen de contar cosas del mundo real.

Definición

El conjunto ℕ

Números enteros positivos, incluyendo el cero en la mayoría de definiciones modernas. Sin parte decimal ni negativos.

ℕ = { 0, 1, 2, 3, 4, 5, 6, ... }
Son infinitos por la derecha: siempre hay un número mayor.
Ejemplos

En la vida cotidiana

Los usamos para contar personas, objetos, días del mes, posiciones en una lista…

0 1 7 100 1.000.000

No son naturales: −3, 1/2, 2.5, √2

Operaciones

¿Qué podemos hacer?

3 + 5 = 8 (natural). 4 × 7 = 28 (natural). El resultado siempre pertenece a ℕ.
5 − 3 = 2 ✓ (natural) 3 − 5 = −2 ✗ — ya no es natural. Por eso necesitamos ampliar a ℤ.
8 ÷ 4 = 2 ✓ 7 ÷ 2 = 3.5 — no es natural. Para esto necesitamos ℚ.
ℤ
01 — Enteros

Números Enteros

Extendemos los naturales hacia el lado negativo. Incluyen temperaturas, deudas, altitudes bajo el mar.

Definición

El conjunto ℤ

ℕ ampliado con los negativos. ℕ está incluido en ℤ: todo natural es entero.

ℤ = { ..., −3, −2, −1, 0, 1, 2, 3, ... }
ℕ ⊂ ℤ — todo natural es entero, pero no al revés.
Usos reales

¿Para qué sirven los negativos?

−15°C Piso −2 Deuda −200€ −3200m altitud
(+) × (+) = (+) → 3 × 4 = 12 (−) × (−) = (+) → (−3)(−4) = 12 (+) × (−) = (−) → 3 × (−4) = −12 Mismo signo: positivo. Distinto: negativo.
Distancia de n al cero. Siempre positivo. |−7| = 7 |4| = 4 |0| = 0
Ahora sí: la resta

Resta siempre posible en ℤ

Restar es sumar el opuesto:

3 − 8 = 3 + (−8) = −5 −4 − (−2) = −4 + 2 = −2
ℤ sigue sin cerrar en división: (−7) ÷ 2 = −3.5
ℚ
02 — Racionales

Números Racionales

Fracciones y decimales exactos o periódicos. Cualquier número que pueda escribirse como p/q con q ≠ 0.

Definición

El conjunto ℚ

Todo número expresable como cociente de dos enteros, con denominador distinto de cero.

ℚ = { p/q | p, q ∈ ℤ, q ≠ 0 }
1/2 −3/4 0.75 0.333… 7
Todo entero es racional: 7 = 7/1. ℕ ⊂ ℤ ⊂ ℚ.
Decimales racionales

Exactos y periódicos

Todo racional tiene expansión decimal que termina o que se repite con un período fijo:

1/4 = 0.25 (exacto) 1/3 = 0.333... (puro) 1/6 = 0.1666... (mixto) 5/7 = 0.714285714285...
Si el denominador simplificado solo tiene factores 2 y/o 5, el decimal es exacto. 1/8 → denom = 2³ → exacto: 0.125 1/6 → denom = 2×3 → periódico: 0.1666...
Operaciones con fracciones

Suma, resta, producto y cociente

1. Busca el mcm de los denominadores. 2. Amplifica cada fracción. 3. Suma solo los numeradores. 1/4 + 2/6 → mcm(4,6) = 12 = 3/12 + 4/12 = 7/12
Numerador × numerador, denominador × denominador. 3/4 × 2/5 = 6/20 = 3/10
Multiplica por la inversa del divisor. 3/4 ÷ 2/5 = 3/4 × 5/2 = 15/8
ℚ está cerrado en las 4 operaciones (÷0 excluido siempre).
𝕀
03 — Irracionales

Números Irracionales

No pueden expresarse como fracción. Su decimal no termina ni se repite. Ejemplos: √2, π, e.

Definición

¿Qué es un irracional?

Un número real que NO puede expresarse como p/q. Su expansión decimal es infinita y no periódica.

√2 = 1.41421356237... π = 3.14159265358... e = 2.71828182845... √3 = 1.73205080756...
Raíces y cuadrados perfectos

Cuándo una raíz es irracional

La raíz cuadrada de n es racional solo si n es un cuadrado perfecto.

√4 = 2 ✓ √9 = 3 ✓ √2 ≈ 1.414… ✗ √5 ≈ 2.236… ✗
Busca factores cuadrados perfectos: √12 = √(4×3) = 2√3 √50 = √(25×2) = 5√2 √72 = √(36×2) = 6√2
Solo se suman si tienen el mismo radical: 3√2 + 5√2 = 8√2 ✓ √3 + √2 — no simplifica √8 + √2 = 2√2 + √2 = 3√2 ✓
Sorpresas matemáticas

Irracional × irracional puede ser racional

Resultado contraintuitivo: dos irracionales pueden cancelarse.

√2 × √2 = 2 (¡natural!) π + (−π) = 0 (¡entero!) √2 × √8 = √16 = 4
No asumas que irracional + irracional = irracional.
Además de irracionales, π y e son "trascendentes": no son raíz de ningún polinomio con coeficientes enteros. Los más famosos de la matemática.
ℝ
04 — Reales

Números Reales

La unión de racionales e irracionales. Cubren toda la recta numérica sin dejar ningún hueco.

Definición

El conjunto ℝ

ℝ = ℚ ∪ 𝕀. Representa todos los puntos de la recta numérica.

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ 𝕀 ⊂ ℝ
Suma, resta, producto y división (sin ÷0) siempre dan un real.
Propiedad fundamental

Completitud de ℝ

ℝ no tiene huecos. Entre dos reales cualesquiera siempre hay otro real.

[a, b] → cerrado: incluye a y b (a, b) → abierto: excluye a y b [a, b) → semiabierto (−∞, b] → todos los reales ≤ b [a, +∞) → todos los reales ≥ a Ejemplo: x ≥ 3 → [3, +∞)
√(−1) no existe en ℝ. Para eso se crean los números complejos ℂ, con la unidad imaginaria i. Es el siguiente paso.

De menor a mayor

Cada conjunto está "dentro" del siguiente. Los irracionales son la parte de ℝ que está fuera de ℚ.

ℕ Naturales: 0, 1, 2, 3, …
ℤ Enteros: …, −2, −1, 0, 1, 2, …
ℚ Racionales: fracciones y decimales periódicos
𝕀 Irracionales: √2, π, e, √3, …
ℝ Reales: todo lo anterior unido
Cadena: ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ y 𝕀 ⊂ ℝ.
ℚ ∩ 𝕀 = ∅ (no comparten ningún elemento).
